# How to Use This Template

## สำหรับอาจารย์ / TA

1. สร้าง GitHub repo ใหม่จากไฟล์ template นี้ หรือ upload โครงนี้เข้า repo ที่ต้องการ
2. เพิ่มนิสิตเป็น collaborator หรือให้นิสิต fork ตามรูปแบบที่กำหนด
3. ให้นิสิตทำ Practical Exam ตาม rubric ในไฟล์ Excel
4. ตรวจจาก README link, evidence file, commit link และ issue link

## สำหรับนิสิต

ให้ทำตามขั้นตอน:

1. เข้า repo ที่อาจารย์กำหนด
2. เปิด README.md
3. แก้ตาราง Team Members เพิ่มชื่อ ทีม role และ evidence link ของตนเอง
4. สร้างไฟล์ `docs/evidence/[github-username].md`
5. เขียน reflection
6. commit ด้วย message ที่อ่านเข้าใจ
7. สร้าง issue 1 รายการ
8. ส่ง link ใน Teams หรือ MS Form
